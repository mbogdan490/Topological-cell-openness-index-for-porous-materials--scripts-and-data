After the deposition is published, replace `[ZENODO_URL]` with the Zenodo record URL (e.g. `https://zenodo.org/records/XXXXXXXX`) and add the DOI line from the record page.

## Using the archive

1. Download `datasets.tar.gz` from Zenodo.
2. Extract it at the root of this repository (same layout as used by the scripts in `codes/`):

   ```bash
   tar -xzf datasets.tar.gz
   ```

3. Run the reproduction scripts as described in the repository README.
